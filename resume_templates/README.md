# 简历模板目录（resume_templates/）

把你认可的**简历模板**（`.md` 文件）放进本目录，OfferClaw 在生成
「项目 → 简历项目经历段」时会把它们作为**格式范例**学习——输出会严格
贴合模板的格式、语气与排版模式。

- 支持多个模板（最多取前 3 个注入）；
- 文件名随意（`README.md` 除外）；
- 目录为空时使用内置默认模式（经典三段式：定位一句话 + 动词开头要点 + 量化成果）；
- 本目录除本说明外**不入 git**（个人文件）。

> 模板里最有用的部分是「项目经历」段的写法范例：
> 项目名/角色/时间的排法、要点 bullet 的句式、量化的呈现方式。
