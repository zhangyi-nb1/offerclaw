## 分块结果

- **源文档**：`user_profile.md`，4960 字符，188 行
- **分块数**：16 块
- **chunk_size**：500 字符，**chunk_overlap**：50 字符
- **分块策略**：按 Markdown 标题层级（`## ` / `### `）→ 分隔线 → 空行 → 换行 → 中文标点

1. 调大 `chunk_size` 到 800–1000，减少碎片
2. 过滤 < 50 字符的无效块
3. 按 Markdown 标题作为天然边界，不跨章节合并

## 检索测试

4 组查询均返回 top-2 结果，链路完整。
