# OfferClaw · 每日执行与复盘（Daily Log）

> 本文件用于承接 JVS Claw 的定时任务：早提醒、晚复盘、动态修正。
> 每一天**追加**一条记录（不要覆盖历史）。格式保持一致，便于 OfferClaw 读取与对比。

---

## 使用约定

- 每天新增一个 `## YYYY-MM-DD` 二级标题区块
- 区块内字段缺失时保留 `【待补充】`，不要删除条目
- 学习留痕内容（笔记 / 链接 / 题目结果）直接粘贴在"学习留痕"区
- OfferClaw 会在晚间复盘时自动生成"偏离度判断"与"明日建议"
- 每周日 OfferClaw 读取最近 7 条记录做周度复盘
- 任何时候都不要删除历史记录，旧的就是数据

---

## 今日主线标签（封闭集合，必填一项或多项）

每天的"今日计划"必须显式标一个主线标签。可选值只能是下面 5 个之一，允许多选（最多 2 个），用 `/` 分隔：

| 标签 | 含义 | 典型动作 |
|---|---|---|
| `补技能` | 通过学习/练习弥补能力缺口 | 看官方文档、做练习题、跟教程 |
| `补项目` | 产出或推进可直接写进简历的项目 | 编码实现、调试功能、补 README |
| `补面试` | 面试准备相关 | 算法题、八股、项目复述、模拟面试 |
| `岗位调研` | 摸清目标岗位的要求与信息 | 搜集 JD、读招聘页、整理目标公司清单 |
| `投递准备` | 简历、自我介绍、作品集整理与投递 | 改简历、拍作品、写 cover letter、实际投递 |

**规则**：
- 不允许自创标签，也不允许"其他"
- 不允许同一天 2 个以上
- 如果一天的实际动作和标签严重不一致（如标了 `补项目` 但全天在刷题），晚间复盘时 OfferClaw 会在"偏离度判断"里指出
